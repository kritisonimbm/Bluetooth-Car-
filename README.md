# Bluetooth Controlled Car (Arduino)

## Objective
To design and implement a Bluetooth-controlled robotic car using Arduino.

## Components Used
- Arduino UNO
- Bluetooth Module (HC-05 / HC-06)
- Motor Driver
- DC Motors
- Jumper Wires

## Working
Mobile se Bluetooth command aata hai.
Arduino us command ke hisaab se motors ko control karta hai.

## Learning
- Bluetooth communication
- Motor control
- Arduino programming
